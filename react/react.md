## Part 1 : 블로그 제작 & 기초 문법

### 기초 문법

- JSX 문법
    1. html 에 class 넣을 땐 className 을 사용한다.
    2. 변수를 html에 넣을 때는 {중괄호} 를 사용
    3. html 에 style 속성을 넣고 싶으면 `<div style={ {color : 'blue', fontSize : '30px'} }> 글씨 </div>` { 속성명 : '속성값' } 방식으로 넣어야된다.
